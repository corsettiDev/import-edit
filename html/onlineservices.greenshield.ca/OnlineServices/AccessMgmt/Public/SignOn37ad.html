Exciting News

Plan Admin portal has arrived!

A new application that will eventually replace Online Admin (OA), Plan Admin was designed to deliver an improved user experience with a Google-like search feature for Plan Members, an easier workflow, and some enhanced capabilities such as the ability to add HCSA/PSA contribution when adding a new Plan Member.

Explore Plan Admin for yourself and discover how easy it is to navigate. Just click the link below and use the same User ID and password you use to log in to Online Admin – it’s that simple. https://adminportal.greenshield.ca/

Plan Admin Cheat Sheet:
  1. Plan Admin is very user friendly; you can start by typing a name or a Plan Member ID in the search bar.
  2. You can find the Admin Guide under the “My Account” icon on the top right of your screen.
  3. The old and new application can coexist. In other words, all changes made in Plan Admin can be seen in OA and vice versa.
Contact your GreenShield Account Team for more information about the new Plan Admin portal.
